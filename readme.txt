== Description ==

### Awesome Custom Scrollbar 

Awesome Custom Scrollbar is a jQuery custom scrollbar for your wordpress website. This plugin will enable awesome custom scrollbar. You can change scrollbar color, border radius, scroll speed, width, hide delay & other settings.

### The Features

1. simple installation and activation, it works with NO modification of your code. (some exceptions can happen, so you can write to me)
2. very stylish scrollbars, with no occupation on your window (original browser scrollbars need some of page space and reduces window/div usable width)
3. you can style main document scrollbar (body) too!! (not all script implements this feature)
4. on all browsers you can scroll: dragging the cursor, mouse wheel (speed customizable), keyboard navigation (cursor keys, pagup/down keys, home/end keys)
5. scroll is smooth (as modern tablet browsing), speed is customizable
6. zoom feature
7. hardware accelerated scroll (where available)
8. animation frame support for smoth scrolling and cpu-saving
9. dragging scroll mode with scrolling momentum (as touch device)
10. tested for all major browsers desktop and mobile versions
11. support for touch devices
12. support for multi-input devices (IE10 with MSPointer)
13. compatible with many other browsers, including IE6, Safari on Mac and WP7 Mango!
14. very customizable aspect of bar
15. native scroll events are working yet
16. fully integrated with jQuery
17. compatibile with jQuery UI, jQuery Touch, jQuery Mobile

== Installation ==

1. Upload 'wpchandra-awesome-custom-scrollbar' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Change Option to Setting->WPChandra Custom Scrollbar Page
4. Change options and injoy!
